发展经济学｜附录A2-技术变化的分类

发表于2023-05-24|更新于2023-11-07|发展经济学笔记
|阅读量:

技术变化的倾向和替代弹性，决定资本-劳动比率（K/LK/LK/L）和工资-租金比率（w/rw/rw/r）之间的关系，进而决定劳动和资本的收入份额。

技术变化通常根据使用生产要素的倾向分类。希克斯分类 (John Hicks, 1932)，以特定资本-劳动比率下劳动和资本的边际替代率的变化方向为基础。

在两要素经济中，希克斯的技术变化可以分为三种类型。

2. 希克斯中性技术进步

**中性技术进步：**对于特定的资本-劳动比率 K/LK/LK/L，技术进步不改变劳动和资本的边际产出的比率 FL/FK=w/rF_L / F_K = {w} / {r}FL​/FK​=w/r。

如图所示：等产量线移动 i0→i1i_0 \to i_1i0​→i1​ 表示技术进步。

劳动和资本的边际生产率为边际技术替代率（marginal ratio of technical substitution）：

\begin{align} MRTS_{LK} = \frac {\partial K} {\partial L} = \frac {F_L} {F_K} \end{align}

其中，MRTSLKMRTS_{LK}MRTSLK​ 即为等产量线 iii 的切线斜率的绝对值。由原点发出的射线 OAOAOA 与两条等产量线 i0i_0i0​、i1i_1i1​ 分别相交于点 aaa、点 bbb，且两点处等产量线的斜率相同，因此为希克斯中性的技术进步。

希克斯中性的技术进步表明，技术进步不改变各要素的边际产出，因此不会改变各要素的边际替代关系 MRTSLKMRTS_{LK}MRTSLK​。

在均衡的竞争性要素市场中，劳动 LLL 和资本 KKK 的边际技术替代率 MRTSLKMRTS_{LK}MRTSLK​ 等于工资与租金比率，即 MRTSLK=w/rMRTS_{LK} = {w} / {r}MRTSLK​=w/r。如果资本-劳动比率不变，技术进步就是中性的。此时，劳动收入 wLwLwL 与资本收入 rKrKrK 的比率不变，意味着劳动和资本之间的收入份额不变。

3. 希克斯劳动节约型技术进步

**劳动节约型技术进步：**对于特定的资本-劳动比率 K/LK/LK/L，技术进步会减小劳动和资本的边际产出的比率 MRTSLK=FL/FK=w/rMRTS_{LK} = F_L / F_K = {w} / {r}MRTSLK​=FL​/FK​=w/r，等成本线发生非平行移动，此时，劳动的边际产出 FLF_LFL​ 下降，资本的边际产出 FKF_KFK​ 升高，应该减少劳动投入，增加资本投入。

**特征：**资本-劳动比率 K/LK/LK/L 越高，即切点与原点的连线越陡峭（比较 OAOAOA 线与 OBOBOB 线），等产量线向原点移动幅度越大。

**结果：**随着技术进步 i0→i1i_0 \to i_1i0​→i1​：

  • 对于固定 K/LK/LK/L 比率的 OAOAOA 线，其交点由 aaa 移动到 bbb，此时切点斜率下降，即劳动与资本的边际产出比率 FL/FKF_L / F_KFL​/FK​ 下降。
  • 对于固定 w/rw/rw/r 比率的 OBOBOB 线，其交点由 aaa 移动到 ccc，K/LK/LK/L 比例提升。

希克斯劳动节约型技术进步可以被定义为对于给定 w/rw/rw/r 诱发 K/LK/LK/L 比例提高的技术进步，资本收入份额的增加是以牺牲劳动收入份额下限为代价的。

结合图解，若在上图中作出两条过 aaa 与 bbb 点的垂线，交横轴于 TaT_aTa​ 与 TbT_bTb​，那么可以得到 TaP0OTa<TbP1OTb\frac {T_aP_0} {OT_a} \lt \frac {T_bP_1} {OT_b}OTa​Ta​P0​​<OTb​Tb​P1​​，即资本收入份额相对侵占了劳动收入份额。

4. 希克斯资本节约型技术进步

**资本节约型技术进步：**对于特定的资本-劳动比率 K/LK/LK/L，技术进步会增大劳动和资本的边际产出的比率 MRTSLK=FL/FK=w/rMRTS_{LK} = F_L / F_K = {w} / {r}MRTSLK​=FL​/FK​=w/r，等成本线发生非平行移动，此时，劳动的边际产出 FLF_LFL​ 升高，资本的边际产出 FKF_KFK​ 下降，应该增加劳动投入，减少资本投入。

**特征：**资本-劳动比率 K/LK/LK/L 越低，即切点与原点的连线越陡峭（比较 OAOAOA 线与 OBOBOB 线），等产量线向原点移动幅度越大。

**结果：**随着技术进步 i0→i1i_0 \to i_1i0​→i1​，对于固定 K/LK/LK/L 比率的 OAOAOA 线，其交点由 aaa 移动到 bbb，此时切点斜率变大，即劳动与资本的边际产出比率 FL/FKF_L / F_KFL​/FK​ 变大。

希克斯资本节约型技术进步可以被定义为对于给定 w/rw/rw/r 诱发 K/LK/LK/L 比例下降的技术进步，劳动收入份额的增加是以牺牲资本收入份额为代价的。

结合图解，若在上图中作出两条过 aaa 与 bbb 点的垂线，交横轴于 TaT_aTa​ 与 TbT_bTb​，那么可以得到 TaP0OTa>TbP1OTb\frac {T_aP_0} {OT_a} \gt \frac {T_bP_1} {OT_b}OTa​Ta​P0​​>OTb​Tb​P1​​，即劳动收入份额相对侵占了资本收入份额。

补充：其他技术进步分类方式：

**哈罗德 (Harrod, 1948) ：**以资本边际生产率 www 表示的资本产出比率 K/YK/YK/Y 的变化为基础

  1. 中性技术进步：K/YK/YK/Y 保持不变
  2. 资本节约型技术进步：K/YK/YK/Y 下降
  3. 劳动节约型技术进步：K/YK/YK/Y 上升

**索洛 (Solow, 1970) ：**以工资率 www 表示的劳动产出比率 L/YL/YL/Y 的变化为基础

  1. 中性技术进步：L/YL/YL/Y 保持不变
  2. 资本节约型技术进步：L/YL/YL/Y 上升
  3. 劳动节约型技术进步：L/YL/YL/Y 下降

三种技术进步分类方式的关系：

  1. 哈罗德中性 ⇒\Rightarrow⇒ 希克斯劳动节约型
  2. 索洛中性 ⇒\Rightarrow⇒ 希克斯资本节约型
