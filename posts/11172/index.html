一、需求理论与供给理论

（一）需求理论

1. 需求量与需求定理

需求量：在给定价格水平下，愿意并能够购买的该商品数量。

需求定理：其他因素不变时，需求量随价格反方向变动。

2. 需求与影响因素

需求：消费者在一定时期内在各种可能的价格水平下愿意而且能够购买的该商品数量。==> 有效需求：有购买意愿且有支付能力

需求的影响因素：

  • 该商品的价格
  • 消费者收入
  • 其他相关商品价格
  • 消费者偏好
  • 预期的未来价格

3. 需求函数与需求曲线

需求函数：Qd=f(pd,po,I,pe,pr)Q_d=f(p_d,p_o,I,p_e,p_r)Qd​=f(pd​,po​,I,pe​,pr​)

pdp_dpd​：该商品的价格

pop_opo​：其他相关商品价格

III：消费者收入

pep_epe​：消费者预期未来价格

prp_rpr​：消费者偏好

4. 需求/需求量的变动

需求的变动：非价格因素（p0,I,pe,prp_0,I,p_e,p_rp0​,I,pe​,pr​）导致的变动，体现为需求曲线的水平移动，如 D1D_1D1​ 曲线移动至 D2D_2D2​ 或者 D3D_3D3​。

需求量的变动：该商品价格变化导致的变化，体现为点在需求曲线上的移动，即 Qd=f(pd)Q_d=f(p_d)Qd​=f(pd​)，如 AAA 点移动到 BBB 点。

（二）供给理论

1. 供给量与供给定理

供给量：在给定价格水平下，愿意而且能够生产的产品数量。

供给定理：其他不变时，供给量随价格同方向变动。

2. 供给与影响因素

供给：生产者在一定时期内在各种可能的价格下愿意而且能够提供出售的该种商品数量。==> 有效供给：有生产欲望且有供给能力

供给的影响因素：

  • 该商品的价格
  • 生产成本
  • 生产技术水平
  • 其他相关商品价格
  • 生产者对未来的预期

3. 供给函数与供给曲线

供给函数：Qs=f(ps,T,po,pe,w,r,...)Q_s=f(p_s,T,p_o,p_e,w,r,...)Qs​=f(ps​,T,po​,pe​,w,r,...)

psp_sps​：该商品的价格

TTT：生产技术水平

pop_opo​：其他相关商品价格

pep_epe​：生产者预期未来价格

www：工人工资

rrr：利率

4. 供给/供给量的变动

供给的变动：非价格因素（T,po,pe,w,r,...T,p_o,p_e,w,r,...T,po​,pe​,w,r,...）导致的变动，体现为供给曲线的水平移动，如 S1S_1S1​ 曲线移动至 S2S_2S2​ 或者 S3S_3S3​。

供给量的变动：该商品价格变化导致的变化，体现为点在供给曲线上的移动，即 Qs=f(ps)Q_s=f(p_s)Qs​=f(ps​)，如 AAA 点移动到 BBB 点。

二、均衡价格与弹性理论

（一）均衡价格

1. 含义

均衡价格为需求量与供给量相等时的价格水平。假设需求量为 Qd=−α+βPQ_d=-\alpha+\beta PQd​=−α+βP，供给量为 Qs=−δ+γPQ_s=-\delta+\gamma PQs​=−δ+γP，那么由 Qd=QsQ_d=Q_sQd​=Qs​ 可以计算出均衡价格。

2. 应用

2.1 支持价格/最低限价

定义：政府为了扶持某一行业的生产而规定的该行业产品的最低价格。支持价格一般高于市场决定的均衡价格。

目的：通常是为了扶持某些行业的发展（如农业）。

为维持支持价格可以采取的措施：

  • 收购过剩商品
  • 对该商品的生产实行产量限制
  • 对购买者给予补贴

2.2 限制价格

定义：政府为了限制某些商品的价格而规定的低于市场均衡价格的最高价格。（传送门：“最高限价的无谓损失”）

**目的：**抑制某些产品的价格上涨，尤其是为了应对通货膨胀。

不良影响：

  • 供不应求会导致消费者排队抢购和黑市交易盛行；
  • 生产者可能粗制滥造，降低产品质量。

为限制价格政府需要采取的措施：

  • 通过配给制等方式控制需求量（如：生产配额）
  • 限定消费者购买数量（如：粮票、肉票、布票）

2.3 “谷贱伤农”的经济学解释

在农业生产活动中，存在着一种经济现象：在丰收的年份，农民的收入反而减少了，这种现象在我国民间被形象地概括为“谷贱伤农”。

造成“谷贱伤农”这种经济现象的根本原因在于：农产品的需求量变动带来的收入增加无法弥补价格下降导致的收入减少损失，其需求价格弹性往往小于 1 。

（二）弹性理论

1. 弹性的含义

因变量对自变量变动的反应程度/敏感程度，一般可以定义为：

\begin{align} 弹性系数=\frac {因变量的变动比例} {自变量的变动比例} \end{align}

也可以写作：

\begin {align} e=\frac {\Delta Y /Y} {\Delta X /X} =\frac {\Delta Y} {\Delta X} \times \frac X Y \end {align}

弧弹性的公式：e=ΔYΔX×XYe =\frac {\Delta Y} {\Delta X} \times \frac X Ye=ΔXΔY​×YX​

弧弹性的中点公式：e=ΔYΔX×(X1+X2)/2(Y1+Y2)/2e =\frac {\Delta Y} {\Delta X} \times \frac {(X_1 + X_2)/2} {(Y_1 + Y_2)/2}e=ΔXΔY​×(Y1​+Y2​)/2(X1​+X2​)/2​（使用中点值替代端点值）

点弹性的公式：e=dYdX×XYe =\frac {d Y} {d X} \times \frac X Ye=dXdY​×YX​

2. 需求价格弹性

2.1 计算方法

需求量与价格之间往往呈反比关系，因此在需求价格弹性取了负号。

需求价格弧弹性：ed=−ΔQdΔP×(P1+P2)/2(Qd1+Qd2)/2e_d =- \frac {\Delta Q_d} {\Delta P} \times \frac {(P_1 + P_2)/2} {(Q_{d1} + Q_{d2})/2}ed​=−ΔPΔQd​​×(Qd1​+Qd2​)/2(P1​+P2​)/2​

需求价格点弹性：ed=−dQddP×PQde_d =- \frac {d Q_d} {d P} \times \frac P {Q_d}ed​=−dPdQd​​×Qd​P​

2.2 需求价格弧弹性的分类

根据弧弹性的大小，可以将商品的需求量与价格的关系分为以下 5 类：

  • 富有弹性：∣ed∣>1|e_d|>1∣ed​∣>1，需求量变化的幅度比价格变化的幅度更大，需求弹性比较平缓
  • 单位弹性：∣ed∣=1|e_d|=1∣ed​∣=1，需求曲线为双曲线 Q=aPQ=\frac a PQ=Pa​ 的情况下，弧弹性处处相等且均为 1
  • 缺乏弹性：0<∣ed∣<10<|e_d|<10<∣ed​∣<1，需求量变化的幅度比价格变化的幅度更小，需求曲线比较陡峭
  • 完全弹性：∣ed∣=∞|e_d| = \infty∣ed​∣=∞，在给定价格下，需求量源源不断
  • 完全无弹性：∣ed∣=0|e_d| = 0∣ed​∣=0，价格变动对需求量没有影响

2.3 线性需求曲线的需求价格弹性

假设线性需求函数为：Qd=a−bPQ_d=a-bPQd​=a−bP，那么需求曲线的斜率为 −1b-\frac 1 b−b1​。

我们关注需求价格弹性的绝对数值，因此不再考虑负号（区别于"需求收入弹性"），上述线性需求函数的需求价格点弹性为：

\begin {align} e_d= b \times \frac P Q= \frac {bP } {a-bP} \end {align}

当 ed=1e_d=1ed​=1 时，bP=a−bPbP=a-bPbP=a−bP，可以算出此时 P=a2b,Q=a2P=\frac a {2b},Q=\frac a 2P=2ba​,Q=2a​，即需求函数的 CCC 点为单位弹性，也就是说：

  • 在中点 CCC 点（线段中点）处，需求价格弹性为 ed=1e_d=1ed​=1；
  • 在中点 CCC 点的下方，AAA 点与 BBB 点的需求价格弹性 ed<1e_d<1ed​<1；
  • 在中点 CCC 点的上方，DDD 点与 EEE 点的需求价格弹性 ed>1e_d>1ed​>1。

理解补充：

在线性需求函数中，可以从以下方面理解点的位置与点弹性大小的关系：

  • 线性需求函数上，价格和需求量呈线性变化关系。
  • 线段 ACACAC：价格 PPP 较小，需求量 QQQ 较大，此时价格下降 1 单位，价格变动的比例相对较大，需求量变动比例相对较小（即弹性公式 e=ΔQ/QΔP/Pe=\frac {\Delta Q / Q} {\Delta P /P}e=ΔP/PΔQ/Q​ 的分母更大），因此弹性小于 1；
  • 线段 CECECE：价格 PPP 较大，需求量 QQQ 较小，此时价格下降 1 单位，价格变动的比例相对较小，需求量变动比例相对较大（即弹性公式 e=ΔQ/QΔP/Pe=\frac {\Delta Q / Q} {\Delta P /P}e=ΔP/PΔQ/Q​ 的分子更大），因此弹性大于 1。

2.4 需求价格弹性的影响因素

  • 商品的可替代性
  • 商品用途的广泛性
  • 商品对消费者生活的重要程度
  • 商品的消费支出在消费者预算总支出中所占的比重
  • 所考察的消费者调节需求量的时间

3. 需求收入弹性

3.1 计算方法

需求量与收入之间一般呈正相关系，因此不取负号（区别于"需求价格弹性"）。

需求收入弹性的计算方式：eM=ΔQΔM×MQe_M=\frac {\Delta Q} {\Delta M} \times \frac M QeM​=ΔMΔQ​×QM​，或者写成点弹性：eM=dQdM×MQe_M =\frac {d Q} {d M} \times \frac M QeM​=dMdQ​×QM​

3.2 商品分类

  • 正常品：eM>0e_M>0eM​>0，随着收入的增加，消费者对正常商品的需求量会上升

    • 奢侈品/高档品：eM>1e_M>1eM​>1，属于富有弹性（需求收入弹性）的商品
    • 必需品：0<eM<10<e_M<10<eM​<1，属于缺乏弹性（需求收入弹性）的商品，如食品消费

  • 劣等品：eM<0e_M<0eM​<0，随着收入的增加，消费者对劣等商品的需求量会下降

恩格尔定律：在一个家庭或一个国家中，食品支出在收入中所占的比例随着收入的上升而下降，即国家或家庭越富裕，食品支出的收入弹性就越小。

4. 需求交叉弹性

4.1 定义与计算方法

定义：在一定时期内，一种商品的需求量变动对于它的相关商品的价格变动的反应程度。

商品 X 的需求量与商品 Y 的价格相关，设 QX=f(PY)Q_X=f(P_Y)QX​=f(PY​)，那么：

需求交叉价格的弧弹性公式：eXY=ΔQXΔPY×PYQXe_{XY}=\frac {\Delta Q_X} {\Delta P_Y} \times \frac {P_Y} {Q_X}eXY​=ΔPY​ΔQX​​×QX​PY​​

需求交叉价格的点弹性公式：eXY=dQXdPY×PYQXe_{XY}=\frac {d Q_X} {d P_Y} \times \frac {P_Y} {Q_X}eXY​=dPY​dQX​​×QX​PY​​

4.2 需求交叉弹性与商品分类

根据需求交叉弹性，两种商品的关系可以分为以下三种类型：

  • 替代品：eXY>0e_{XY}>0eXY​>0，表示提高商品 Y 的价格能提高商品 X 的需求量，∣eXY∣|e_{XY}|∣eXY​∣ 越大，替代性越强；
  • 互补品：eXY<0e_{XY}<0eXY​<0，表示提高商品 Y 的价格会降低商品 X 的需求量，∣eXY∣|e_{XY}|∣eXY​∣ 越大，互补性越强；
  • 不相关商品：eXY=0e_{XY}=0eXY​=0，表示商品 X、商品 Y 是相对独立的两种商品。

5. 供给价格弹性

5.1 计算方法

供给量与价格之间往往呈正比关系。

供给价格弧弹性：es=ΔQsΔP×PQse_s =\frac {\Delta Q_s} {\Delta P} \times \frac P {Q_s}es​=ΔPΔQs​​×Qs​P​

供给价格点弹性：es=dQsdP×PQse_s = \frac {d Q_s} {d P} \times \frac P {Q_s}es​=dPdQs​​×Qs​P​

5.2 供给价格弹性的分类

根据供给价格弹性的大小，可以将商品的供给量与价格的关系分为以下 5 类：

  • 富有弹性：es>1e_s>1es​>1，供给量变化的幅度比价格变化的幅度更大
  • 缺乏弹性：0<es<10<e_s<10<es​<1，供给量变化的幅度比价格变化的幅度更小
  • 单位弹性：es=1e_s=1es​=1，供给量变化幅度与价格变动程度相等，是一种极端情况
  • 完全弹性：es=∞e_s = \inftyes​=∞，供给曲线为一条水平线，这是商品出现严重过剩时的情况
  • 完全无弹性：es=0e_s = 0es​=0，供给曲线为一条垂直线，价格变动对供给量没有影响，一般是极其稀缺、珍贵或无法复制的商品。

5.3 供给价格弹性的影响因素

  • 生产的难易程度
  • 生产规模与规模变化的难易程度
  • 成本的变化
  • 时间的差异、厂商的生产能力、对未来价格的预期

6. 蛛网模型

6.1 简介

蛛网模型运用弹性原理解释某些生产周期较长的商品在失去均衡时发生的不同波动情况的一种动态分析理论。

基本假设：

  • 供给有时滞：商品的本期产量取决于前一期的价格，即供给函数为：Qts=f(Pt−1)Q_t^s=f(P_{t-1})Qts​=f(Pt−1​)
  • 需求无时滞：商品本期的需求量取决于本期的价格，即供给函数为：Qtd=f(Pt)Q_t^d=f(P_{t})Qtd​=f(Pt​)

使用方程表示：

  • 供给有时滞：Qts=−δ+γPt−1Q_t^s=-\delta+\gamma P_{t-1}Qts​=−δ+γPt−1​

  • 需求无时滞：Qtd=α−βPtQ_t^d=\alpha-\beta P_tQtd​=α−βPt​

  • 本期供需均衡：Qts=QtdQ_t^s=Q_t^dQts​=Qtd​

因此，可以求得 PtP_tPt​ 与 Pt−1P_{t-1}Pt−1​ 的关系：

\begin{align} P_t = - \frac \gamma \beta P_{t-1} + \frac { \alpha + \delta } {\beta} \end{align}

令期初价格为 P0P_0P0​，那么：

\begin {equation} \begin {split} P_t & = \left( - \frac {\gamma} {\beta } \right) ^ t \cdot P_0 + \left\{ \frac { \alpha + \delta } {\beta} \left[ 1+ \left(- \frac {\gamma} {\beta } \right)^1 + \left(- \frac {\gamma} {\beta } \right)^2+ ...+\left(- \frac {\gamma} {\beta } \right)^{t-1} \right] \right\} \\\\ & = \left( - \frac {\gamma} {\beta } \right) ^ t \cdot P_0 + f \end {split} \end {equation}

我们将上式花括号设为 fff 单独计算。其中，方括号内为一个首项为 a0=1a_0=1a0​=1、公比为 q=−γβq=- \frac {\gamma} {\beta}q=−βγ​ 的等比数列，可以使用等比数列的求和公式求解。

因此，蛛网模型主要包括三种情况。

6.2 蛛网模型的 3 种情况

对于某一种商品，供给价格弹性与需求价格弹性的相对大小关系不同，商品的价格均衡情况会有所不同。

6.2.1 收敛型蛛网：供给弹性小于需求弹性

当 t→∞t \to \inftyt→∞，如果 γβ<1\frac \gamma \beta<1βγ​<1，那么 Pt→PeP_t \to P_ePt​→Pe​，也就是说随着时间 ttt 的增加，若 γ<β\gamma < \betaγ<β，供给曲线比需求曲线更陡峭（斜率更大），供给弹性小于需求弹性。此时，供给变动对价格变动的反应程度小于需求变动对价格变动的反应程度，此时，价格最后自发地趋向于均衡水平。

  • 需求价格弹性：ed=−dQddP×PQde_d =- \frac {d Q_d} {d P} \times \frac P {Q_d}ed​=−dPdQd​​×Qd​P​

6.2.2 发散型蛛网：供给弹性大于需求弹性
